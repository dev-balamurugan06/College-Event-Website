# 🎉 College Event Website

A modern and responsive **College Event Management Website** designed to showcase events, manage registrations, and provide real-time updates for students.

---

## 🚀 Features

* 📅 Event Listings (Technical / Cultural / Workshops)
* 📝 Online Registration System
* ⏰ Event Schedule & Timings
* 📢 Announcements & Updates
* 📱 Fully Responsive Design
* 🎯 Clean and User-Friendly UI

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript *(or React / Next.js if used)*
* Backend: Node.js / Flask *(if applicable)*
* Database: MongoDB / MySQL *(if used)*
